Welcome to Vault Web Clipper

Follow these steps to connect the extension to your Obsidian vault.

Step 1: Install "Local REST API" in Obsidian

Go to Obsidian Settings > Community Plugins > Browse > Search "Local REST API" > Install & Enable.

Step 2: Get your API Key

Go to Obsidian Settings > Community Plugins > Local REST API. Copy the API Key shown there.
